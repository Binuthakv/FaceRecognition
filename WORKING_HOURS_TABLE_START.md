## 🔍 QUICK FILTER GUIDE

### Filter 1: Date Range
```
Step 1: Click Start Date field
Step 2: Pick date (e.g., 2024-01-01)
Step 3: Click End Date field
Step 4: Pick date (e.g., 2024-01-31)
Step 5: Click "Apply Filters"

Result: Shows records for January 2024
```

### Filter 2: User ID
```
Step 1: Click User ID dropdown
Step 2: Select user (e.g., john@company.com)
Step 3: Click "Apply Filters"

Result: Shows all of John's records
```

### Filter 3: Combined
```
Step 1: Set Start Date: 2024-01-15
Step 2: Set End Date: 2024-01-31
Step 3: Select User: john@company.com
Step 4: Click "Apply Filters"

Result: Shows John's records for Jan 15-31
```

## ❓ FAQ

**Q: Where does the data come from?**
A: UserWorkingHours table (calculated by working hours service)

**Q: Can I filter by multiple users?**
A: No, but you can filter by date range to see all users

**Q: How often is data updated?**
A: When working hours are calculated (via API service)

**Q: Can I export the data?**
A: Currently view only (export feature coming soon)

**Q: What if I see no records?**
